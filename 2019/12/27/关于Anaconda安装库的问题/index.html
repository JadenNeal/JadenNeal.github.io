关于Anaconda安装库的问题

2019-12-27

于是进入prompt卸载重装matplotlib库，结果是卸载成功，重新安装却失败了。

弹出了“拒绝访问”的错误。

第一种：“ERROR 13 “拒绝访问” ”：

这种情况只要右键管理员模式 运行promt即可。

再次检查conda命令，在prompt命令行下输入“conda install matplotlib”。——————失败

使用“pip3 install matplotlib”————————————————————失败

进入Navigator，进入base环境下，搜索matplotlib，安装，——————失败

重启电脑，再次尝试“conda install matplotlib”，成功。

此时输入conda list，发现matplotlib确实重新安装成功了。
